# 깃허브 명령어 정리

### git init

- 저장소 생성

### git add .

- 저장소에 추가

### git commit -m "메세지"

- 커밋

### git push origin main(master)

- 커밋내용을 서버에 업로드

### git pull origin main(master)

- 서버 저장소의 변경 내용을 현재 디렉터리에 가져오기

### git remote reomve origin

- origin이라는 원격 저장소를 지우기

### git remote add origin "원격 저장소"

- origin이라는 이름으로 원격 저장소를 연결
